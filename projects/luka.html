Luka, AI agent for recrutiment agencies

Luka is a SaaS platform that leverages AI to help recruitment agencies win more clients — from sourcing and prospecting to outreach across WhatsApp, LinkedIn, and email. It also enables AI-powered message personalization to make every interaction more effective.

Technical stack

I fully developed the SaaS in MVP mode. It features an intuitive interface and an AI agent that detects recruitment signals and contacts the right people with AI-assisted automation — a semi-autopilot approach.

Key technical details:

  • ‍Frontend: Built with Next.js for a fast and responsive user experience.
  • Backend: Developed in Node.js, handling cron jobs and background tasks efficiently.
  • AI Orchestration: Leveraged LangChain to manage and orchestrate AI workflows.
  • Payment & Auth: Integrated Stripe for payments and OAuth for secure authentication.
  • Data Management: High-volume data handled with a NoSQL database for scalability.
  • Deployment: Hosted on an EC2 server for reliability and performance.This setup allows the platform to automate lead detection and engagement while remaining intuitive and scalable.
